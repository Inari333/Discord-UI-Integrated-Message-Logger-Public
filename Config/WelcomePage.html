Welcome to Discord Integrated UI Message Logger!

Instructions

Use Discord normally or leave it open to the channel you want to log. The logger will automatically capture messages in real-time.

To views manage logs or settings, click the extension icon. We recommand to pin it.

If you need to dispute a message with Discord customer service, provide the message id by simply right-click on it and select "Copy Message ID".
If the message is deleted, you can hover your mouse above the message or the username to view the message id.
